cluster 集群方式

安装

  1. 下载代码

    mkdir -p /apps/redis-cluster & cd /apps/redis-cluster
    wget http://download.redis.io/releases/redis-5.0.3.tar.gz
    tar zxvf redis-5.0.3.tar.gz
    cd redis-5.0.3
    make
    cp ./src/redis-server ../
    cp ./src/redis-cli ../

  2. 创建一组配置文件

mkdir 9001 9002 9003 9004 9005 9006

每个目录下写入配置

# 设定端口
port 9001
# 开启集群
cluster-enabled yes
# 保存节点配置文件
cluster-config-file "nodes.conf"
# 集群节点的超时时限,默认值为 15000
cluster-node-timeout 5000
# 日志文件
logfile "log.log" 
# 开启数据持久化
appendonly yes
# 数据文件
dbfilename "dump.rdb"
# 守护进程
daemonize yes
# 节点间同步密码
masterauth 123456
# 设置密码
requirepass 123456

每个目录下都执行一遍

cd ./9001
../redis-server ./redis.conf

创建集群

--replicas 1 表示,每个主节点创建一个从节点
下面操作,将会将配置应用到各个节点,并(join)起来,让节点们互相通讯起来

./redis-cli --cluster create 127.0.0.1:9001 127.0.0.1:9002 127.0.0.1:9003 127.0.0.1:9004 127.0.0.1:9005 127.0.0.1:9006 --cluster-replicas 1

连接方式

redis-cli -c -p 9001 -a 123456

package main

    "net/http"
    "fmt"

欢迎使用 MWeb

2019/1/16

首先介绍一下 MWeb 是什么，MWeb 是专业的 Markdown 写作、记笔记、静态博客生成软件。MWeb 使用的是 Github Flavored Markdown 语法，在使用 MWeb 前，建议你一定要去 MWeb 官网首页 看一下介绍视频，MWeb 官网也做了比较详细的帮助，建议你也看一下大概内容，帮助的网址为：http://zh.mweb.im/help.html。

需要您重点注意的

MWeb 是有两种模式的：外部模式和文档库模式。MWeb 为了满足所有 Markdown 使用需求，设计了两种模式！外部模式用于打开和编辑所有本地 Markdown 文档。另外为了方便用 Markdown 记笔记，MWeb 设计了文档库模式，文档库中的文档也支持一键生成静态博客，以便于分享。关于两个模式的情况，还是建议您去看一下官网的视频和帮助。
